SpringBoot脚手架项目，完成项目的初始化搭建，使你更快进入业务开发

完成的功能：
1. token认证
2. SpringSecurity的配置（加密方式设置，cors设置等）
3. redis的序列化配置（使用jackson，未使用fastjson）
4. swagger的配置（可以在页面中使用token）
5. 日志的配置
6. Mybatis的分页配置
7. 统一异常捕获处理

各依赖版本
1. SpringBoot: 2.7.10
2. Mybatis-Plus: 3.5.3.1
3. Swagger: 3.0
使用的依赖均通过checkmarx的监测，无漏洞（暂时）
